How to play.

Navigation

  • Use arrow keys to move along corresponding directions.
  • You can only move freely on sand blocks.
  • You cannot move outside the grid.

Breaking Blocks

  • To break a wooden block, keep pressing the key used to face it.
  • The number of hits to break a block is specified in game creation.
  • The block does not store hits - if you turn away from hitting a block, it 'resets'.
  • Concrete blocks cannot be broken.

Points accumulation

  • Points are gotten from broken blocks. 1 block = 1 point.
  • If two gamers try to break a specific block, whoever breaks it first gets the point.
  • The gamers with the most blocks broken win.
  • The first positions rank up.

Cheats

  • Pursue a player to reduce their points, try breaking the blocks they want to break first.
  • Try to be the game creator, you will not see the usernames of all gamers if you join last
